# ngx-simple-table

Angular version 12.1.0

## Demo

[Stackblitz](https://stackblitz.com/edit/hm-ngx-simple-table)

## Features

The table has basic features as below:

- pagination (from backend)
- check item
- check all items in a page
- check all items in all pages
- fixed header
- fixed scrollbar
- resizable columns
